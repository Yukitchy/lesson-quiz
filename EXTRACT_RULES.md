# 板書md→クイズ語彙の抽出ルール（夜間ルーチン用）

レッスン記録mdの「## 3. 文法・表現ポイント」「## 4. 語彙」系セクション（見出し表記ゆれに柔軟対応）から、4択クイズに使える語彙アイテムを抽出する。

## 対象生徒
- 日本語を学ぶ継続生徒のみ。**Yuri（ネイティブ雑談）・そうた/Sota（英語トレセン・別アプリ）・単発の体験レッスンは対象外**（→ `merge_vocab.py --skip`）
- 体験レッスンでも、その後継続が決まって config.json に居る生徒なら対象にしてよい

## 抽出
- 基本形: `- 共産主義（きょうさんしゅぎ）／communism` → ja=共産主義, yomi=きょうさんしゅぎ, en=communism
- 読み仮名・英訳が欠けた行は、一般的な単語なら自分の知識で自信を持って補完してよい。固有名詞・「※要確認」付き・不確かなものはスキップ
- 文法セクション内でも「〜に依存する (to depend on ~)」のような出題可能な語彙・慣用表現は拾う。活用表・助詞の説明などの純粋な文法解説は拾わない
- フルセンテンスの会話パターン（「趣味は何ですか？」等）は不採用。中の単語が語彙として立つなら単語だけ採用
- en が埋められないアイテムは入れない（クイズにならない）。カタカナ語は yomi=null でOK
- d はレッスン日（ファイル名の YYYY-MM-DD）

## マージ後
- 1件以上マージしたら `python3 build_quiz.py` → git commit & push（該当生徒のページだけ変わる。slug=URLは config.json 固定で不変）
- 4語未満の生徒は build_quiz.py が自動スキップする（語数が増えれば自動で生成解禁）ので特別扱い不要
